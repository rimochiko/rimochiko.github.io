CSS彩虹字+海浪效果+彗星Loading

又隔了一天，然后继续学习张老师的codepen作品，其中一个是很奇幻的彩虹字，还有一个是超级美丽透心凉的海浪效果，另一个是一个彗星拖着尾巴的Loading动画。（突然发现日志已经被CSS开头文章占领了）。

彩虹字

See the Pen Stripy rainbow text effects by Comehope (张偶) (@comehope) on CodePen.

效果解析

这个彩虹字一开始并不好分析，它有很层，虽然我们第一个想到的是linear-gradient，但是这里的那个颜色的变化，每一条都又把红橙黄绿都变了一头，如果我们要写全渐变动画好像也挺麻烦的，重复的动画写N种咯，把位置调一调。

<span data-text="web"></span>
<span data-text="web"></span>
<span data-text="web"></span>
<span data-text="web"></span>

这是HTML结构，实际上又和原来某个做过的使用的技巧差不多，先把本身的“WEB”给隐藏了，靠伪类来实现这个效果。这样我们就有8个WEB，一个WEB有两种颜色，一共可以有16种颜色。
把他们重叠在一起，用z-index来控制层次关系。当然啦这样肯定是只能显示最上面这个，所以它给每个都设置了一个高度，所以这样大家都可以露一部分出来。
这又让我联想到了昨天那个分裂的文字，因为我们设置高度只能露出最上面的部分，所以它要能够显示的话，z-index最小的那个一定是越高的。
另外这里又用到了CSS变量，height: calc(100% - 10% * var(--n))，然后在各个里面设置–n的值，这样写确实感觉方便了不少，不过感觉还是很多人不太支持用CSS变量去写。

海浪效果

See the Pen Waves by Comehope (张偶) (@comehope) on CodePen.

效果解析

这个效果真的超级好看！！
可以看到这个海浪是由三层组成的，所以HTML结构也有三个wave，我们只要做出一个波浪，剩下的设置它的延迟效果和颜色就可以做出这种层次的感觉。
（然后我就想不出要怎么做了，于是去查看了老师的源码来研究）。
发现他竟然是用一个超级大的正方形，进行一个360°的旋转，并且还给这个正方形加上圆角化（不能是50%这样就是一个圈圈在转，那我们也看不出效果了）。后期再加一点透明度，就有通透的感觉了。
我不禁感叹，这样的想法真的太有想法啦>_<！厉害厉害

彗星Loading

See the Pen Comet rotating loader by Comehope (张偶) (@comehope) on CodePen.

效果解析

很容易想到是用border-radius来实现，但是注意的是下半部分应该设置为transparent，要不然就会出现像月牙一样的形状。另外这个尾巴的定位标准，老师是从圆心到末端发出了一条线，然后小球以这个线为基准定位的，我在想如果直接把尾巴以整个盒子去定位的话，确实不太好定位，因为端点是在对角线上——左右上下角还要再内侧一点，所以并不好处理。另外设置animation-direction: reverse;可以让它往反方向动哦！
