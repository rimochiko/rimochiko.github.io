CSS立体书+技能表+流动框

瞅瞅上次这个系列，竟然已经有快1个月了，我震惊.jpg！所以这次我一下就做了三个练习，一个是三本立体书，一个是技能表，还有一个是类似于之前的一个流动框效果。另外在这里继续宣传一下原作者[1][张老师]。

立体书

See the Pen book by rimochiko (@mochilee) on CodePen.

解析效果

经过多次的练习，其实咱发现，做一个东西没头绪的时候，就去一个个分解它。这个书的话，如果是平面的我们肯定很容易做出来，但是它有一个立体的效果。这个立体的效果其实就是侧边一个平行四边形+上面一个平行四边形组成的。这里的话刚好两块，用::before和::after就可以实现啦！

实现步骤

首先做出书的封面，加一个渐变效果，然后给span标签（也就是书标题）部分加上样式，这个都比较基础啦了，没有什么很新奇的地方。
然后给它加上书的侧边，设置为absolute相对于书的封面，同样的上到下的渐变，另外加上一个filter:brightness(num)的滤镜，就是把颜色变暗，这样给人感觉有光影效果。然后让它成为倾斜的，加上transform:skewY(45deg)，这个是在Y轴上产生一个倾斜度。
类似的方法，把上面的平行四边形写出来就差不多啦!
