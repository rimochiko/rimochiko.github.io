vue.js安装&初次体验

我终于有机会学习vue.js了~
此次vue.js的安装，我是跟着官方的文档来完成的，
感觉比暑假初次体验react.js相比，要方便太多了。
本地直接先：

npm install vue

之后安装vue的大家族~（也是跟着官方文档来的）

npm install -g vue-cli

装好后就可以初始化一个项目

vue init webpack my-project

然后进入这个新建的项目文件夹，这时候我就碰到了一个问题。
init自动生成了pakage.json，我们可以直接下载依赖，
但是npm install实在太慢了，下载东西下载半天都没反应。
