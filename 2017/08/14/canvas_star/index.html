小结~Canvas制作星星闪耀效果

今晚听了慕课网老师的Canvas制作星星效果的课程，颇有收获，来小小总结一下。

其中重要的API

1.创建和获取canvas画布：

can = document.getElementById('canvas');
ctx = can.getContext("2d");

这个比较初级，算是回忆一下。

中心方法、属性

1.刷新画布的方法：
常用的有三种requestAnimFrame()，setTimeout()和setInterval()。
但是在这里，老师建议我们用第一种，setTimeout()和setInterval()本质上是差不多的，它会根据我们写的时间间隔去执行，而requestAnimFrame()执行的时间间隔不相同，它会根据电脑运行实际情况去充分调用，所以效率会更高。但是它存在兼容问题，所以可以用下面这个函数尽可能解决问题

window.requestAnimFrame = (function() {
    return window.requestAnimationFrame || window.webkitRequestAnimationFrame || window.mozRequestAnimationFrame || window.oRequestAnimationFrame || window.msRequestAnimationFrame ||
        function( /* function FrameRequestCallback */ callback, /* DOMElement Element */ element) {
            return window.setTimeout(callback, 1000 / 60);
        };
})();

2.drawImage()
它有9个参数，分别是：图片对象，图片X位移，图片Y位移，图片截取宽度，图片截取高度，图片在canvas的X位置，图片在canvas的Y位置，图片在canvas里的宽，图片在canvas里的高。利用这个函数我们可以很方便的实现动画序列帧变化。

3.globalAlpha
设置全局的透明度，如果我们只想透明部分，就要搭配save()和restore()函数一起。

starObj.prototype.draw = function(){
    ctx.save();
    ctx. = life;
    ctx.drawImage(starPic,this.picNo*7 ,0 , 7 , 7 , this.x,this.y,7 ,7 );
    ctx.restore();
}

这个例子就是只把星星透明度改变，而不改变背景。

项目实现的原理

我们希望能够随机出现很多星星，我们就先创建星星对象，再创建n个星星实例，随机分配给它们为位置坐标和移动的速度。
当它们移出屏幕时就被赋予新的实例，也就是又小时候又重现了。

项目注意的地方

因为我们采用requestAnimFrame()，所以动画的时间间隔会不一样，我们就需要自己去处理一下获取到这个间隔，
然后再通过自己写一个判断来估计这个时间来执行画面刷新。
另外还添加了一个鼠标移入效果才运行，移出效果又消失，判断鼠标的位置，为了兼容性我们可以这样判断：

if(e.offsetX || e.layerX ) {
    var px = e.offsetX == undefined? e.layerX:e.offsetX;
    var py = e.offsetY == undefined? e.layerY:e.offsetY;
}

以上就是此次听课的小总结。
