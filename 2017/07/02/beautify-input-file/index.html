美化file类型的input+上传图片预览

之前一个月可以说是经历了很多考试，还有一次惊心动魄的学校实训QAQ，
可以说是特别忙，不过也因为经历了这次实训，也还算是学到了很多，毕竟第一次做了后台，完成了一个整的网站。

然后回到这个话题下面先放一个预览

这里有几个要思考的问题：
1.file类型的input我们并不能直接美化它，所以我们需要做一个代替它的容器：
即点击这个容器后触发“点击上传”~
为了更好的美化预览，写出如下CSS样式：

#file-upload（上传input）
{
    display: none;
}

#file-btn（代替input触发点击的容器）
{
    width: 200px;
    height: 100px;
    background-size:auto 100%;
    background-position: center center;
    background-repeat: no-repeat;
    background-image: url(../images/upload.jpg);  /*为没有添加图片时的默认背景*/
    border: 1px dashed #aaa;
    cursor: pointer;
}

2.关于图片上传预览~一般情况下我们是无法通过得到用户电脑上的路径来调用（为了保护隐私），
在这里我使用的方法是用编码读取，即base64编码。

function photoPreview(file)
{
   var prevDiv = document.getElementById('file-btn');

   if (!/.(gif|jpg|jpeg|png|GIF|JPG|png|BMP|bmp)$/.test(file.value))
   {
           return false;
   }

   if (file.files && file.files[0]) {
      var reader = new FileReader();
      reader.onload = function(evt) {
        prevDiv.style = "background-image:url(" + evt.target.result + ")"; 
      }
      reader.readAsDataURL(file.files[0]);
    } else {
        prevDiv.style = "background-image:url(" + file.value + ")"; 
    }    
}

这段代码本来是在网上搜索到的，然后修改了一点。
